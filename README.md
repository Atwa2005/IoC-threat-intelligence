# IoC Extraction & Threat Intelligence Enrichment

## Features
- Extract IP addresses using Regex
- Extract URLs
- Extract Domains
- Extract File Hashes
- Enrich IoCs using VirusTotal API
- Calculate Risk Severity
- Generate JSON Report

## Requirements

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
